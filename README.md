# WebUrl

A URL class for HTTP and HTTPS, extracted from OkHttp and tracking their 3.x branch.


## Usage

```
<dependency>
  <groupId>net.disy.oss</groupId>
  <artifactId>weburl</artifactId>
  <version>2.0</version>
</dependency>
```
